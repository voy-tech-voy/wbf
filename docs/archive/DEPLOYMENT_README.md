# ImgApp - Standalone Version

## What's included:
- `dist/ImgApp.exe` - Single-file standalone executable (~60 MB)
- `Launch_ImgApp.bat` - Easy launcher script

## How to run:
1. **Double-click** `Launch_ImgApp.bat` to start the application
2. Or navigate to `dist/` folder and double-click `ImgApp.exe`

## Distribution:
- **Single File**: Just copy `dist/ImgApp.exe` to distribute
- **Size**: Approximately 60 MB (includes Python runtime and all dependencies)
- **No installation required**: Runs on any Windows 10+ machine

## Features:
- Image conversion (JPG, PNG, WEBP, BMP, TIFF)
- Video conversion (MP4, AVI, MOV, etc.)
- GIF creation from videos
- Multiple quality variants
- Batch folder processing
- Resize options (by ratio or pixel width)
- Dark theme interface
- Drag & drop support
- Start/Stop conversion control

## Requirements:
- Windows 10 or later
- FFmpeg (should be installed in system PATH for video conversions)
- No Python installation required

## Distribution:
This standalone executable can be copied to any Windows computer and run without installing Python or any dependencies. It's a completely self-contained application.
